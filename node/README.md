# Setup
```sh
npm init
npm install -g pnpm@latest-10
npx pnpm add @types/node typescript nodemon ts-node -D
npx pnpm create @eslint/config@latest
npx pnpm add express ajv axios dotenv
npx pnpm add @types/express -D
```
